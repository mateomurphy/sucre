# Usage

<!-- usage -->
```sh-session
$ npm install -g sucre
$ sucre COMMAND
running command...
$ sucre (-v|--version|version)
sucre/0.1.0 darwin-x64 node-v11.14.0
$ sucre --help [COMMAND]
USAGE
  $ sucre COMMAND
...
```
